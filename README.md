# Credit Card Fraud Detection Using Machine Learning

## Project Overview

This project focuses on identifying **fraudulent credit card transactions** using machine learning methods. Fraud detection plays a vital role in financial systems as it helps protect users and organizations from financial losses.

## Dataset

creditcard.csv

The dataset includes transaction details with anonymized features, transaction amounts, and a class label indicating whether a transaction is legitimate or fraudulent.

## Objectives

* Explore and analyze the transaction dataset
* Perform data preprocessing and normalization
* Develop a machine learning model for fraud detection
* Assess the model’s performance using evaluation metrics
* Present insights through data visualization

## Steps Performed

1. Imported and examined the dataset
2. Analyzed the distribution of fraudulent and normal transactions
3. Applied feature scaling to relevant variables
4. Divided the data into training and testing sets
5. Trained the model using the Logistic Regression algorithm
6. Evaluated the model using accuracy, confusion matrix, and classification report

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

## Results

The trained model was able to detect fraudulent transactions with satisfactory accuracy, demonstrating the usefulness of machine learning techniques in fraud detection tasks.

## Conclusion

This project highlights how machine learning can be effectively applied to monitor and identify suspicious financial transactions, contributing to stronger and more secure financial systems.
